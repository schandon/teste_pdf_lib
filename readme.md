# Inicio de Testes
Antes de qualquer coisa esse código é pra testar funcionalidade de escrever num pdf especifico, então vamos aos passos.
### Passo 01
* Vamos utilizar o comando de buildar para gerar o arquivo do tipo .js 
```pnpm run build```

``` pnpm dev```
